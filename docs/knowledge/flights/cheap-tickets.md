# 如何买到便宜机票

## 最佳购票时间

国际机票通常提前2-3个月预订价格最优。周二和周三出发的航班往往比周末便宜。

## 比价平台推荐

Skyscanner、Google Flights和Kayak等聚合平台可以一次性对比多家航空公司的价格。天巡网（Skyscanner）覆盖最广。

## 促销活动提醒

关注航空公司的会员促销和双十一、黑五等购物节的机票折扣。航空公司的会员日通常会有特价票放出。

## 里程积分技巧

累积航空里程可以兑换免费机票或升舱。选择合适的常旅客计划和信用卡可以加速积分积累。
