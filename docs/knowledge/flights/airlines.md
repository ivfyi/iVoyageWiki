# 各大航空公司对比

## 国内航空公司

国航、南航、东航和海航覆盖国内主要国际航线。国航的欧洲航线网络最完善，南航在大洋洲航线上有优势。

## 亚洲航空公司

全日空、新加坡航空和国泰航空常年位列全球最佳航空公司前列。服务品质出色但价格相对较高。

## 欧洲航空公司

汉莎航空、法航和荷兰皇家航空是欧洲的主要航司。瑞安航空和易捷航空是欧洲最大的廉价航空公司。

## 廉价航空对比

亚航、捷星和春秋航空是亚洲知名廉航。廉航通常不包含餐饮和行李，购买时需计算附加费用。
