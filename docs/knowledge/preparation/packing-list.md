# 行李打包清单

## 证件类

护照及复印件、签证、身份证、驾照及翻译件、旅行保险单、机票和酒店预订确认单。

## 衣物类

根据目的地气候准备衣物。基本原则是层叠穿搭，准备一件防水外套和舒适的步行鞋。

## 电子设备

手机、相机、充电器、转换插头、移动电源和备用存储卡。记得下载离线地图以备不时之需。

## 洗漱用品

旅行装洗漱用品、防晒霜、驱蚊液和个人护理用品。液体需符合航空随身携带规定。

## 药品类

常用药品包括感冒药、止痛药、肠胃药、创可贴和晕车药。慢性病患者需备足处方药并携带医嘱。
