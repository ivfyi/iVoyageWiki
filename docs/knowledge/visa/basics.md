# 签证基础知识

## 什么是签证

签证是主权国家颁发给外国公民的入境许可证明。它通常以贴纸或电子形式附在护照上，标明允许入境的期限和目的。

## 常见签证类型

旅游签证、商务签证、学生签证、工作签证和过境签证是最常见的类型。旅行者通常只需要申请旅游签证即可。

## 签证申请基本流程

一般流程包括：确定签证类型 → 准备材料 → 在线填写申请表 → 预约递交时间 → 递交材料和指纹 → 等待审核 → 领取护照。

## 签证费用

各国签证费用从免费到数百美元不等，通常与签证类型、处理速度和双边协议有关。
