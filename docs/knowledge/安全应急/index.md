# 🛡️ 安全与应急指南2026

### 出行前准备

- 查询并保存中国驻目的地大使馆的**地址和电话**
- 记下**旅行保险公司24小时救援电话**（重要！）
- 下载**领事直通车APP**，获取领事保护信息
- 在**中国领事服务网**注册"出国及海外中国公民自愿登记"

## 人身安全

### 出行安全

- 不要深夜独自外出，特别是偏僻地区和不安全街区
- 不要乘坐**黑车**，优先选择正规出租车或网约车（Uber、滴滴等）
- 走路时不要玩手机，注意看路和周围环境
- 随身携带贵重物品，不要放在背包外侧口袋
- 不要接受陌生人的**食物、饮料和香烟**（可能含迷药）
- 手机地图提前查好路线，避免在路边掏手机查路

### 住宿安全

- 入住后**反锁房门**，拉上防盗链
- **不要给陌生人开门**，确认身份后再开门
- 贵重物品存入**保险箱**（酒店房间内或前台）
- 离开房间时**关好门窗**，即使只是出去一会儿
- 熟悉酒店的**安全出口和消防通道**位置
- 选择有24小时前台和监控的住宿

### 饮食安全

- 不要在路边无卫生保障的小摊吃东西
- **不要喝生水**，喝瓶装水或烧开的水
- 海鲜**煮熟后再吃**，避免食物中毒
- 选择**客人多、翻桌快**的餐厅，食材新鲜有保障
- 肠胃敏感的人建议随身携带肠胃药

## 财产安全

- **现金和银行卡分开存放**，放不同位置
- 不随身携带**大量现金**（够当天用即可）
- 不在公共场所**露财**（数钱、展示贵重首饰等）
- 手机和钱包放在**贴身口袋**，背包背在前面
- 购物时注意看管好自己的物品（试衣服时包不离身）
- ATM取钱时注意遮挡密码，检查机器是否有异物（盗刷卡装置）
- 护照和钱包分开存放，避免一起丢失

## 常见问题应急处理

### 护照丢失

1. **立即报警**：前往最近的警察局报案，取得**报警回执**（补办证件的必需材料）
2. **联系使领馆**：前往中国驻外使领馆申请**补发护照**或**旅行证**
3. **准备材料**：提供2张证件照、护照复印件（如有）、身份证明
4. **临时措施**：如需紧急回国，申请**旅行证**（办理更快，仅限回国使用）
5. **后续处理**：回国后到户籍所在地公安局补办新护照

> **预防措施**：随身携带护照复印件+电子扫描件，护照和钱包分开存放

### 钱包/手机丢失

1. **立即挂失**：打电话给银行挂失银行卡，联系运营商挂失手机卡
2. **报警处理**：前往警察局报案，取得报警回执
3. **联系银行**：补办银行卡或让家人汇款应急
4. **联系运营商**：补办手机卡
5. **远程操作**：通过"查找我的iPhone"或"查找设备"定位/锁定手机，远程擦除数据
6. **联系家人**：告知情况，让家人汇款到备用账户

### 生病/受伤

1. **轻微不适**：自行服用随身携带的常用药，多休息多喝水
2. **严重情况**：立即拨打**急救电话**（120或当地急救号码）
3. **就医流程**：
   - 前往正规医院就诊
   - 保留所有**医疗单据**（病历、处方、发票）
   - 翻译病历和处方（如果需要）
4. **联系保险公司**：拨打保险救援电话，告知就医情况，获取理赔指引
5. **后续理赔**：回国后按保险公司要求提交理赔材料

### 遇到诈骗

1. **保持冷静**，不要惊慌，不要被对方催促做决定
2. **立即报警**：拨打当地报警电话
3. **不要转账**：绝不向陌生人汇款或转账
4. **保留证据**：保存聊天记录、通话录音、转账凭证等
5. **联系银行**：如果已经转账，立即联系银行尝试冻结账户
6. **向使领馆求助**：重大诈骗案件可联系中国驻外使领馆

### 常见骗局提醒

| 骗局类型 | 手法 | 防范方法 |
|---------|------|---------|
| 街头换钱 | 以优惠汇率吸引换钱，用假币或少给钱 | 只去正规银行或ATM换钱 |
| 出租车绕路 | 故意绕远路，打表器做手脚 | 用打车APP，或上车前确认价格 |
| 假警察 | 冒充警察查证件/搜包，趁机偷钱 | 要求看警徽，报警核实 |
| 热情帮忙 | 主动帮买票/指路，索要"服务费" | 礼貌拒绝，自己查信息 |
| 祈福骗局 | 以祈福消灾为名调包财物 | 不要相信陌生人要你拿钱出来 |
| 免费样品 | 塞给你东西后强行收费 | 不接受陌生人递来的任何物品 |

## 实用建议

- **购买旅行保险**：尤其是出国游，医疗保额建议50万以上，包含紧急救援
- **重要证件拍照备份**：存在手机相册和云盘各一份，邮箱也存一份
- **随身带少量现金**：20-100美元/欧元等值当地货币，以备不时之需
- **告知家人行程**：把行程单、酒店信息和紧急联系方式发给家人
- **遇到问题及时求助**：不要独自硬撑，联系使领馆、保险公司或当地警察
- **下载离线地图**：出发前在谷歌地图中下载目的地的离线地图，没网络也能导航
- **记下住处地址**：用当地语言写下来，打车或问路时出示
