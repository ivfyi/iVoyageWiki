# 日本JR Pass

## JR Pass类型

JR Pass分为7天、14天和21天三种有效期。还有地区性通票如关西广域通票和北海道通票。

## 购买方式

JR Pass必须在入境日本前通过海外指定代理商购买换取券，入境后在JR车站兑换正式通票。

## 使用范围

JR Pass覆盖JR集团旗下的新干线（除希望号、瑞穗号）、特急列车和普通列车。私铁和地铁需另外购票。

## 激活与使用

兑换时指定启用日期。进出站需走人工通道或持有通票的自动闸机，不能通过普通闸机。

## 划算程度分析

东京-大阪往返一次即可回本。行程中包含多个远程城市间的移动时，JR Pass的性价比极高。
