# 🌊 西湖

![西湖全景](https://picsum.photos/seed/hangzhou-westlake/800/400)

> **【一句话定位】** 西湖是中国唯一的世界文化遗产湖泊类景观，适合所有类型的旅行者——亲子漫步、情侣约会、摄影创作、文化探访、休闲放松，每个人都能在这里找到属于自己的乐趣。

## 【核心亮点】

- ★★★★★ **苏堤春晓**——春天苏堤全长2.8公里，桃花垂柳交相辉映，是西湖十景之首
- ★★★★★ **三潭印月**——西湖标志性景观，1元人民币背面图案原型，登岛观景最佳
- ★★★★★ **雷峰夕照**——傍晚登雷峰塔，俯瞰西湖全景+日落，金色湖面如梦如幻
- ★★★★☆ **断桥残雪**——许仙白娘子相会之地，雪景最出名，平日也可远眺保俶塔
- ★★★★☆ **音乐喷泉**——每晚19:00/20:00两场，喷泉+灯光+音乐，免费观赏

## 【最佳游览时间】

| 时段 | 体验 |
|------|------|

| ☀️ **上午 9:00-11:00** | 光线柔和，适合白堤/苏堤散步+坐游船，游客开始增多 |

| 🌙 **晚上 19:00-21:00** | 湖滨灯光亮起，音乐喷泉+夜游西湖，浪漫气氛拉满 |

**⚠️ 避开时间：** 五一/十一假期人山人海；周末下午湖滨人流极大；暑期白天高温暴晒

## 【推荐游览路线】

**经典半日游（3-4小时）**
```
断桥 → 白堤 → 平湖秋月 → 苏堤 → 花港观鱼 → 雷峰塔 → 音乐喷泉
```
- **断桥（20min）**：西湖标志性起点，远眺保俶塔最佳位置
- **白堤（30min）**：1公里长的湖中堤，两侧垂柳依依
- **平湖秋月（15min）**：喝茶赏湖景，秋天看月亮倒映最佳
- **苏堤（40min）**：2.8公里长堤，六桥起伏，春天桃花绝美
- **花港观鱼（30min）**：红鲤鱼成群，可买饲料喂鱼，小朋友最爱
- **雷峰塔（1h）**：登塔俯瞰西湖全景，看"雷峰夕照"
- **音乐喷泉（30min）**：湖滨路看免费喷泉表演

**一日深度游（6-7小时）**
```
断桥 → 白堤 → 孤山（省博） → 苏堤 → 花港观鱼 → 乘船至三潭印月 → 岳王庙 → 雷峰塔 → 音乐喷泉
```

## 【隐藏玩法/小众体验】

- 🚲 **清晨骑行西湖**：早6点前共享单车环湖，全程约10公里，1小时骑完，没有人的西湖完全是另一番景象
- 🛶 **手划船环湖**：在苏堤南口找手划船师傅（约¥150/小时），走内西湖水道，不坐机动船，安静穿过桥洞
- 🍵 **龙井村喝一杯**：从苏堤南口步行20分钟到龙井村，找农家喝杯明前龙井（¥30/杯），看茶农炒茶
- 📖 **孤山脚下西泠印社**：中国最著名的篆刻社团，藏在孤山深处，庭院精致，几乎没人
- 🌸 **乌龟潭小众赏花**：杨公堤附近的乌龟潭，樱花/梨花季人少花美，本地人才知道

## 【避坑提示】

- ❌ **不要在湖边买茶叶**——路边的"龙井茶"大部分是外地茶冒充，去龙井村或正规茶庄买
- ❌ **不要坐湖边揽客的游船**——私人游船价格虚高（开口¥100+），去码头坐官方游船¥55含三潭印月门票
- ❌ **节假日别打车上苏堤**——苏堤周末禁止机动车通行，且周边堵车严重，地铁龙翔桥站最靠谱
- ❌ **雷峰塔不用坐电梯**——排队半小时，走楼梯只需5分钟，还能看每层不同展览
- ✅ **带空水瓶**——西湖边有直饮水点（白堤/苏堤/湖滨），免费接水省钱又环保
- ✅ **下雨天也别取消行程**——"烟雨西湖"才是真正的江南韵味，反而雨天人少更有意境
