# 🏰 宋城

![宋城](https://picsum.photos/seed/hangzhou-songcheng/800/400)

> **【一句话定位】** 宋城是以南宋文化为主题的大型仿古主题公园，以"宋城千古情"歌舞秀闻名，适合家庭亲子游、文化体验者、以及喜欢看大场面演出的游客。

## 【核心亮点】

- ★★★★★ **《宋城千古情》演出**——被称为"一生必看"的歌舞秀，以全息投影+真水真雪+空中飞人呈现杭州千年历史，**绝对值回票价**
- ★★★★★ **宋城夜景**——华灯初上，灯笼亮起，整座宋城仿佛穿越回南宋，比白天美十倍
- ★★★★☆ **鬼屋"聊斋惊魂"**——中国最好的鬼屋之一，场景逼真，演员专业，胆小的慎入
- ★★★★☆ **王员外招婿**——经典互动表演，抛绣球选"女婿"，被选中可上台参与，非常欢乐
- ★★★★☆ **古装换装体验**——免费汉服+古装租赁（¥50-100/套），穿上在古城里拍照出片率极高

## 【最佳游览时间】

| 时段 | 体验 |
|------|------|
| 🌅 **上午 9:00-11:00** | 园区刚开门人少，适合先玩室外项目、逛仿古街、拍没人挡的照片 |
| ☀️ **下午 13:00-15:00** | 看日场演出+室内项目，避开正午暴晒 |
| 🌇 **下午 15:00-18:00** | **入园最佳时间！** 逛2小时+看17:00场千古情+等亮灯 |
| 🌙 **晚上 18:00-21:00** | 宋城最美时刻！灯笼全亮，看19:30/20:30场额外演出，夜景拍照绝了 |

**💡 最佳策略：** 下午3点入园 → 逛宋城街+拍照 → 看17:00或19:00场千古情 → 逛夜景 → 看广场表演

**⚠️ 避开时间：** 五一/十一假期人山人海（千古情演出排队长达2小时）；暑期周末下午暴晒

## 【推荐游览路线】

**半日精品游（4-5小时）**
```
宋城东门 → 宋河东街 → 市井街 → 千古情剧院（看演出） → 宋城广场（美食+夜景）
```
- **宋河东街（30min）**：仿南宋街市，各种手工艺作坊、糖画、吹糖人
- **市井街（40min）**：小吃街+古装拍照区+各种互动表演
- **千古情剧院（1h）**：看《宋城千古情》演出（每天14:00/17:00/19:00三场）
- **宋城广场（30min+）**：打铁花表演、篝火晚会、夜景灯光

**一日全覆盖游（6-7小时）**
```
东门 → 宋河东街 → 水上乐园（夏季） → 市井街 → 鬼屋 → 千古情剧院 → 宋城广场 → 夜景拍照
```

## 【隐藏玩法/小众体验】

- 🎭 **免费看《大地震》**——除千古情外，宋城还有一部5D实景剧《大地震》，2008年汶川地震主题，非常震撼，**门票已含**
- 🎪 **街头艺人互动**——下午时段市井街有各种杂耍、皮影戏、糖画，全部免费观看，打赏随意
- 🌊 **夏季水上乐园**——每年6-9月宋城开放水上乐园区域，门票包含在内不需额外付费
- 📜 **活字印刷体验**——宋河东街有个活字印刷工坊，¥20可亲手印一张宋词带回家
- 🏮 **夜游宋城灯会**——每年的元宵节前后、中秋节，宋城会有特别灯会，比日间更华丽

## 【避坑提示】

- ❌ **不要在门口买黄牛票**——宋城门票实名制，黄牛票大概率刷不进去，直接在官网或美团买
- ❌ **千古情演出建议选17:00场**——14:00场太早（赶不上夜景），19:00场看完出来园区快关门了，17:00场看完出来正好逛夜景
- ❌ **别周末去！**——周末人比工作日多3倍以上，排队体验极差，能工作日去尽量工作日
- ❌ **千古情演出别买最便宜的票**——普通席在两侧角落，视野受限，加¥60升贵宾席体验好很多
- ✅ **穿汉服去！**——宋城是最适合穿古装的主题公园，到处都是穿汉服的游客，毫无违和感
- ✅ **雨天也有好处**——雨天人少，打着油纸伞在宋城拍照更有古风意境，而且大部分项目在室内
- ✅ **演出提前20分钟入场**——千古情剧场提前20分钟开始入场，晚了要从侧门进，错过开场
