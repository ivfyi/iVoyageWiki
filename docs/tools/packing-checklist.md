# 🧳 行李打包检查表

> 按类别逐一核对，出发前打勾确认，再也不会忘带东西。下方清单覆盖大多数旅行场景，可根据目的地和季节增删。

## 📋 证件类（最重要！放随身包！）

- [ ] **护照**（有效期6个月以上，检查是否需签证页空白）
- [ ] **签证**（电子签打印件 + 手机存PDF备份）
- [ ] **身份证**（国内游必备，住酒店乘高铁）
- [ ] **机票行程单**（打印件 + 手机离线存）
- [ ] **酒店预订单**（打印 + 手机备份，入境可能查验）
- [ ] **旅行保险单**（打印 + 存手机打印版）
- [ ] **护照复印件**（2份，随身包+行李箱各一份）
- [ ] **证件照**（2张白底，补办证件用，尺寸查好目的地要求）
- [ ] **驾驶证 + 国际驾照翻译件**（如需自驾，有些国家需公证）
- [ ] **学生证 / 老年证 / 教师证 / 军官证**（景点门票优惠，很多国家认ISIC国际学生证）

**💡 证件收纳Tips：** 所有证件装一个文件袋放随身包，手机里存一份扫描件 + 发一份给家人。

## 👕 衣物类

### 通用基础
- [ ] 上衣（按天数×1 + 备2件）
- [ ] 裤子/裙子
- [ ] 内衣裤（按天数×1 + 备2套）
- [ ] 袜子（按天数×1 + 备2双）
- [ ] 外套（根据目的地气温准备1-2件）
- [ ] 睡衣/家居服
- [ ] 舒适运动鞋（每天步行1万+步，千万别穿新鞋！）
- [ ] 拖鞋（国外酒店大多不提供一次性拖鞋）
- [ ] 凉鞋/沙滩鞋（海岛/夏天必备）
- [ ] 帽子（防晒/保暖，视目的地）
- [ ] 腰带

### 🌸 按季节/场景
| 场景 | 必带 |
|------|------|
| 夏季/热带 | 短袖×N、短裤、裙子、防晒衣、速干T恤 |
| 冬季/寒冷 | 羽绒服、保暖内衣、抓绒衣、手套、围巾、暖宝宝、羊毛袜 |
| 雨季/海岛 | 雨伞、防水外套/雨衣、速干衣裤 |
| 滑雪 | 雪服、雪镜、滑雪手套（可当地租，提前确认） |
| 商务/正式场合 | 正装一套、皮鞋、领带/丝巾 |
| 徒步/户外 | 冲锋衣、登山鞋、速干衣、登山杖（可托运） |

### 🎒 按旅行类型
- [ ] **城市游**：轻便为主，1件好搭配的外套 + 舒适步行鞋
- [ ] **海岛度假**：泳衣2套、浮潜装备（如自备）、罩衫/防晒衣
- [ ] **自然/户外**：冲锋衣、徒步鞋、保暖层、速干内衣
- [ ] **多国游**：洋葱式穿衣（多层叠穿），带1件百搭外套

## 🧴 洗漱用品类

- [ ] 牙刷、牙膏（国外酒店大多不提供，尤其是欧洲和日本）
- [ ] 洗发水、护发素、沐浴露（旅行装，或到当地买）
- [ ] 洗面奶、护肤品、化妆品（液体超过100ml需托运！）
- [ ] 毛巾、浴巾（一次性压缩毛巾超方便，占空间小）
- [ ] **防晒霜**（SPF50+ PA+++，热带/海岛/滑雪必备！）
- [ ] 剃须刀 + 剃须膏
- [ ] 卫生巾 / 卫生棉条（部分国家不好买，备足）
- [ ] 隐形眼镜 + 护理液 + 备用框架眼镜
- [ ] 指甲刀、棉签
- [ ] 梳子、发圈、发夹
- [ ] **个人用药**：感冒药、肠胃药、过敏药、晕车药（带原装盒+说明书）
- [ ] 驱蚊液/防蚊贴（东南亚、非洲、南美必备）
- [ ] 漱口水 / 牙线

**💡 液体分装Tips：** 买分装瓶套装（100ml以内），安检轻松过；托运可选大瓶。

## 📱 电子用品类

- [ ] **手机 + 数据线**（多带一根备用线，线最容易坏）
- [ ] **充电宝**（≤20000mAh/100Wh，随身携带！不可托运）
- [ ] 耳机（降噪耳机是长途飞行神器，没有之一）
- [ ] 相机 + 充电器 + 内存卡 + 读卡器
- [ ] **转换插头**（出国必备！不同国家标准不同）
  - 欧标（欧洲、亚洲） / 英标（英、港、新加坡） / 美标（美、日、台） / 澳标（澳、新西兰、中国）
- [ ] 插线板（一个转换插头+插线板，解决所有设备充电）
- [ ] 平板 / 笔记本电脑 + 充电器
- [ ] 智能手表 + 充电器
- [ ] 自拍杆 / 轻便三脚架
- [ ] 备用手机 / 备用SIM卡（如有）
- [ ] GoPro / 运动相机 + 配件（户外/水上活动必备）
- [ ] 手机防水袋（海岛/水上项目必备）

**💡 充电策略：** 带一个2-3口快充头（GaN充电器），少带充电头多带线。

## 💊 药品类

| 类别 | 推荐药品 | 备注 |
|------|---------|------|
| 感冒 | 感康、快克、板蓝根 | 温差大易感冒 |
| 退烧 | 布洛芬、对乙酰氨基酚（必带！） | 儿童用美林/泰诺林 |
| 肠胃 | 蒙脱石散、黄连素、藿香正气水 | 水土不服最常见的 |
| 晕车/晕船 | 茶苯海明（晕海宁）、晕车贴 | 出海/盘山路必备 |
| 抗过敏 | 氯雷他定（开瑞坦） | 海鲜/花粉过敏 |
| 外伤 | 创可贴、碘伏棉签、无菌纱布 | 小伤口及时处理 |
| 防蚊虫 | DEET驱蚊液、蚊虫叮咬膏 | 东南亚/非洲重点防疟疾 |
| 眼药水 | 人工泪液（单支装） | 长时间飞行/戴隐形 |
| 个人处方药 | 备足量 + 带英文处方/医生证明 | 部分国家入境需申报 |

**⚠️ 药品入境注意：** 含麻黄碱（如新康泰克）、可待因（如止咳水）的药物在很多国家被禁，出行前查好。中药/中成药也可能被查扣。

## 💰 财务类

- [ ] **银行卡**（至少2张不同银行，Visa/Mastercard各一）
- [ ] **当地货币现金**（少量，到当地ATM取更划算）
- [ ] 人民币现金（回国备用）
- [ ] 信用卡（选境外消费返现高的）
- [ ] 钱包 / 零钱包（放硬币）
- [ ] 护照财物防盗带 / 隐形腰包（防盗）

**💡 省钱Tips：**
- 境外取现：华夏银行、宁波银行等每月前几笔免手续费
- 刷卡：全币种信用卡免货币转换费
- 现金：不要在机场换汇柜台换，汇率极差

## ✅ 行前准备检查

### 出发前1周
- [ ] 预订机票/火车票（确认时间和航站楼）
- [ ] 预订住宿（确认地址和入住时间）
- [ ] 办理签证（或检查有效期）
- [ ] 购买旅行保险（至少覆盖医疗+行程取消+财物丢失）
- [ ] 预约热门景点门票（很多需提前1-4周）

### 出发前3天
- [ ] 查好当地天气（按天气调整衣物）
- [ ] 下载离线地图（Google Maps离线区域）
- [ ] 开通国际漫游 / 购买eSIM / 当地SIM卡
- [ ] 下载必备APP（翻译、地图、当地交通）
- [ ] 换汇/取当地货币
- [ ] 通知银行出境（避免信用卡被锁）

### 出发前1天
- [ ] 在线值机（提前24小时，选好座位）
- [ ] 称重行李（确保不超重）
- [ ] 收拾随身包（证件+充电宝+外套+水杯）
- [ ] 告知家人行程 + 紧急联系方式
- [ ] 关闭手机不必要的订阅和自动扣费
- [ ] 确认机场交通方式 + 出发时间（留足提前2h到机场）

### 出发当天
- [ ] 再次检查护照和机票
- [ ] 充电宝充满电，手机下载离线内容
- [ ] 关好家中水电气
- [ ] 垃圾桶清空（回来后不会有异味）
- [ ] 带上一颗轻松的心，出发！🎉
