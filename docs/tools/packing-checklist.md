# 行李打包检查表

> 按类别逐一核对，出发前打勾确认，再也不会忘带东西。

## 📋 证件类（最重要！）

- [ ] **护照**（有效期6个月以上）
- [ ] **签证**（确认有效期和停留期）
- [ ] **身份证**（国内游必备）
- [ ] **机票行程单**（打印+手机备份）
- [ ] **酒店预订单**（打印+手机备份）
- [ ] **旅行保险单**（打印+存手机）
- [ ] **护照复印件**（2份，随身+行李箱各一份）
- [ ] **证件照**（2张，白底，补办证件用）
- [ ] **驾驶证+国际驾照翻译件**（如需租车）
- [ ] **学生证/老年证/军官证**（门票优惠）

## 👕 衣物类

### 通用

- [ ] 上衣（按天数+备1-2件）
- [ ] 裤子/裙子
- [ ] 内衣裤（按天数+备2套）
- [ ] 袜子（按天数+备2双）
- [ ] 外套（根据天气）
- [ ] 睡衣
- [ ] **舒适的运动鞋**（每天步行1万+步）
- [ ] 拖鞋（酒店拖鞋不卫生）
- [ ] 凉鞋/沙滩鞋（海岛必备）
- [ ] 帽子（防晒/保暖）
- [ ] 围巾/丝巾

### 按季节

- [ ] **夏季**：短袖、短裤、裙子、防晒衣
- [ ] **冬季**：羽绒服、保暖内衣、手套、围巾、暖宝宝
- [ ] **雨季**：雨伞、防水外套
- [ ] **滑雪**：雪服、雪镜、手套（可当地租）

## 🧴 洗漱用品类

- [ ] 牙刷、牙膏（国外酒店多不提供）
- [ ] 洗发水、护发素、沐浴露（旅行装）
- [ ] 洗面奶、护肤品、化妆品
- [ ] 毛巾、浴巾（一次性压缩毛巾方便）
- [ ] 防晒霜（SPF50+，热带/海岛必备）
- [ ] 剃须刀、剃须膏
- [ ] 卫生巾/卫生棉条
- [ ] 隐形眼镜+护理液
- [ ] 指甲刀、棉签
- [ ] 梳子、发圈、发夹

## 📱 电子用品类

- [ ] 手机+数据线（多带一根备用）
- [ ] **充电宝**（20000mAh以内，随身携带）
- [ ] 耳机（降噪耳机长途飞行神器）
- [ ] 相机+充电器+内存卡
- [ ] **转换插头**（出国必备）
- [ ] 插线板（一个插线板解决所有充电）
- [ ] 手表/智能手表
- [ ] 自拍杆/三脚架
- [ ] 备用手机（如有）

## 💊 药品类

- [ ] 感冒药（感康、快克）
- [ ] 退烧药（布洛芬、对乙酰氨基酚）
- [ ] 肠胃药（蒙脱石散、黄连素、藿香正气水）
- [ ] 晕车药/晕船药
- [ ] 创可贴、碘伏棉签
- [ ] 抗过敏药（氯雷他定）
- [ ] 个人常用药（备足量+带处方）
- [ ] 防蚊虫药（热带地区必备）
- [ ] 眼药水
- [ ] 体温计

## 🧰 其他用品类

- [ ] 雨伞/折叠伞
- [ ] 保温杯/水杯
- [ ] 纸巾、湿纸巾、消毒湿巾
- [ ] 垃圾袋/收纳袋（装脏衣服）
- [ ] 小锁（住青旅用）
- [ ] 购物袋/环保袋
- [ ] 颈枕、眼罩、耳塞（长途飞行三件套）
- [ ] 折叠衣架、晾衣绳
- [ ] 防水袋（装湿衣物）
- [ ] 一次性马桶垫
- [ ] 零食（长途航班/路上充饥）

## 💰 财务类

- [ ] 银行卡（至少2张不同银行，Visa/Mastercard）
- [ ] 当地货币现金（少量）
- [ ] 人民币现金（回国备用）
- [ ] 信用卡（境外消费返现卡优先）

## 🗺️ 行前准备检查

- [ ] 预订机票和住宿
- [ ] 办理签证
- [ ] 购买旅行保险
- [ ] 预订机场接送/查好交通路线
- [ ] 下载离线地图
- [ ] 开通国际漫游/购买SIM卡/eSIM
- [ ] 下载必备APP
- [ ] 告知家人行程和紧急联系方式
- [ ] 关闭手机不必要的订阅和自动扣费
- [ ] 预约景点门票（热门景点需提前）
- [ ] 查看天气预报
- [ ] 确认航班时间和航站楼
- [ ] 在线值机（提前24小时）
