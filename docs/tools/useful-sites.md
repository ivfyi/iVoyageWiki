# 实用网站推荐

## 机票比价

Skyscanner、Google Flights、携程、去哪儿等平台可对比各大航空公司价格。

## 住宿预订

Booking.com、Airbnb、Agoda、Hostelworld 覆盖全球各类住宿。

## 交通查询

Google Maps、Rome2Rio、当地交通APP方便查询路线和时刻表。

## 攻略社区

穷游网、马蜂窝、TripAdvisor、Reddit 相关版块提供真实旅行经验。

## 实用APP

翻译类（Google Translate）、地图类（Google Maps）、汇率类（XE Currency）。
