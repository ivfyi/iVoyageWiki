# 🔗 实用网站和APP推荐

> 全网实测好用旅行工具汇总，覆盖行前规划、出行中、应急处理全场景。

## ✈️ 机票比价与预订

| 平台 | 网址 | 特点 | 推荐理由 |
|------|------|------|---------|
| **Google Flights** | [google.com/travel/flights](https://google.com/travel/flights) | 全球航线最全，价格走势图 | ⭐ 首选！日历视图一眼看出哪天便宜，价格追踪自动发邮件提醒 |
| **Skyscanner 天巡** | [skyscanner.com](https://skyscanner.com) | 支持多城市/模糊搜索"任意地点" | 适合"不知道去哪"时探索低价目的地 |
| **携程** | [ctrip.com](https://ctrip.com) | 国内用户最友好，中文客服 | 国内出发最优，联程机票/机酒套餐多 |
| **去哪儿** | [qunar.com](https://qunar.com) | 比价能力强，低价提示 | 常比携程便宜，尤其国际机票 |
| **Momondo** | [momondo.com](https://momondo.com) | 价格预测准确 | 比Skyscanner好用，UI更清晰 |
| **Kayak** | [kayak.com](https://kayak.com) | 功能全面，价格预测 | 整合多个供应商，价格预测准 |
| **Hopper** | [hopper.com](https://hopper.com) / APP | AI预测机票涨跌 | 告诉你"现在买还是再等"，适合不着急的人 |

**💡 购票策略：** 提前2-3个月买国际票最优，周中（周二周三）价格最低，设置价格提醒等降价。

## 🏨 住宿预订

| 平台 | 网址 | 特点 | 适用场景 |
|------|------|------|---------|
| **Booking.com** | [booking.com](https://booking.com) | 覆盖最广，免费取消 | ⭐ 首选！酒店/民宿/青旅都有，评分真实 |
| **Agoda** | [agoda.com](https://agoda.com) | 亚洲价格最优 | 东南亚、日韩住宿首选，常比Booking便宜10-20% |
| **Airbnb** | [airbnb.com](https://airbnb.com) | 整套房源/当地特色 | 多人出行/长住/想体验当地生活 |
| **Hostelworld** | [hostelworld.com](https://hostelworld.com) | 全球青旅最全 | 背包客首选，评分和评价真实可靠 |
| **HotelTonight** | APP | 当天特价酒店 | 当天入住折扣大，适合说走就走 |
| **Tripadvisor** | [tripadvisor.com](https://tripadvisor.com) | 比较各家价格 | 先看评分再看价格，一个地方比N个平台 |

**💡 省钱策略：** 同一家酒店，Booking和Agoda比价；住3晚以上看Airbnb；旺季提前2个月订。
